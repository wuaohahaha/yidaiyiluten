• 2021年

    12月3日 昆明-万象线开通
    中国昆明站内的复兴号“绿巨人”、老挝万象站内的“澜沧号”动车组列车缓缓开动，开启一段跨越山河、相互奔赴的旅程。

    12月4日 成渝-万象线开通
    成渝两地至老挝的货运时间将缩短至3天左右，大大降低企业的运输成本，货运效率也大幅提高。

    12月8日 南京-万象线开通
    载有76个标箱出口货物的“江苏号”中老铁路国际货运列车启程，是长三角地区开行的首趟中老铁路。

    12月13日 义乌-万象线开通
    首趟列车70个标箱满载小家电、机械配件、纺织用品、钾肥等货物。

    12月27日 玉溪-万象线开通
    首趟列车共有21只集装箱冷链货柜，整合了当地11户企业农产品出口货运需求。

  • 2022年

    1月6日 怀化-万象线开通
    满载怀化冰糖橙、双峰小农机、三一重工小型挖掘机等货物的国际班列从怀化出发后，将向西经过贵阳、六盘水、昆明等城市，从云南磨憨口岸出关后经老挝磨丁到达老挝万象。

    1月10日 澜湄快线开通
    较其他中老国际货运列车具有到货时间可控、通关和运行速度快等特点，是中国昆明与老挝万象间“客车化”的货运服务产品。

    3月2日 江津-万象线开通
    共运载37个集装箱，货值约400万元，包括化工品、建筑材料、通用机械、日用百货等。

    3月30日 宝鸡-万象线开通
    标志着陕西首趟中老铁路国际货运列车正式开行。

    4月21日 敦煌-万象-曼谷线开通
    是甘肃省首次通过中老国际铁路向境外直接出口本地产品，也是国内首发的石棉集装箱国际铁路联运列车。

    4月22日 西安-万象线开通、沈阳-万象线开通
    西安-万象线首次主要运输的是一批纺织品。列车从西安国际港站驶出，经云南磨憨口岸出境，预计5天左右抵达老挝万象。

    5月21日 首趟中老铁路“铁路快通”开通
    “铁路快通”即“铁路进出境快速通关”模式。运用“铁路快通”模式通关，能节约通关时间24小时，缩短整体运行时间1-2天。

    6月23日 首趟中老铁路旅游专列开通
    旅游专列首发仪式在昆明举行，“坐着火车游云南”活动正式开启。

    7月1日 中老泰正式互通
    北向连接中老铁路万象南站货场，南向连接既有老挝米轨铁路，由老中铁路有限公司投资建设，站内设置标准轨换装线1条、米轨换装线2条，可同时换装25辆泰国米轨铁路货车，年换装作业能力约80万吨。

    11月16日 达州-万象线开通
    是西部陆海新通道中老铁路首发，助力达州等地重点企业产品拓展国际市场。

  • 2023年

    4月13日 首列万象-昆明中老铁路国际旅客专列开通
    昆明至万象间可实现乘火车当日通达。

    4月25日 海安—琅勃拉邦国际货运列车化肥专列开通
    首趟列车70个标箱化肥的货运列车从铁路南京货运中心海安货运站启程。

    10月16日 中老中越铁路国际冷链货运班列双向首发
    两列满载着云南特色果蔬的冷链班列，同时从云南省玉溪市研和站驶出。
