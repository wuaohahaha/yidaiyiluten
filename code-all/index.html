在漫长的人类文明交往历史中，贯穿东西、连接欧亚的古丝绸之路熠熠生辉，从商品流通到文化交融，从经济互利到人文互启，在友好交往的和平理念下，沿线各国人民共同谱写出一幅繁荣壮丽的史诗篇章。

2013年秋，习近平总书记提出共建“丝绸之路经济带”和“21世纪海上丝绸之路”，这份共建“一带一路”的倡议，令跨越千年的悠悠驼铃和海上的猎猎破风声再度响起，也让这条象征着开放与发展的欧亚经济大走廊在新时代复苏，成为中国与世界的互利共赢之路。

十年来，在各方的共同努力下，共建“一带一路”始终坚持共商共建共享原则，秉持开放、绿色、廉洁理念，以高标准、可持续、惠民生为目标，取得了实打实、沉甸甸的重大历史性成就。

01

「六廊六路多国多港」彰显中国担当

“一带一路”的共建以互连为基础，其基础设施的“硬联通”建设是区域协同发展的重中之重，因此我国提出“六廊六路多国多港”的主体框架思路，通过10年的践行将理念转化为行动，将愿景转变为现实。当前，“六廊六路多国多港”的基本架构已初具规模，基本形成了“陆海天网”四位一体的互联互通格局，为促进经贸和产能合作、加强文化交流和人员往来奠定了坚实基础。

其中，“六廊”是指中蒙俄、新亚欧大陆桥、中国——中亚——西亚、中国——中南半岛、中巴、孟中印缅六条国际经济合作走廊；“六路”即铁路、公路、水路、空路、管路、信息高速路互联互通路网；“多国”是作为合作重点的若干重要国家；“多港”指代构建若干海上支点的重要港口。“六廊六路多国多港”的建设为建立和加强沿线各国伙伴关系、畅通亚欧大市场起到重要作用。

建设的脚步未曾停歇，十年来，为落实“一带一路”互联互通的整体格局，中国在共建国家承包工程累计签订合同额2万亿美元，完成营业额1.3万亿美元，工程涉及境外工程投融资、设计咨询、设备采购、建设施工、运营管理等多个方面。庞大的工程规划有效降低了共建国家参与国际贸易的成本，提高了沿线各国接入世界经济的能力和水平，激发了区域经济乃至全球贸易的更大发展潜力、更强发展动力。

中国企业在“一带一路”建设中不断打造“质量好、负责任”的闪亮名片，建设起一个个“国家地标”、一项项“民生工程”、一座座“合作丰碑”，为带动中国产品技术服务“走出去”和深化国际产能合作作出贡献，彰显了以和平合作、开放包容、互学互惠、互利共赢为核心的丝路精神。

十年磨剑，于朝夕间争渡，从广袤土地到浩渺海洋，从脚下大陆到头顶星空，共建“一带一路”的倡议在今天已然结出累累硕果。在每一个平凡的日子，上万旅客登上雅万高铁便捷出行，近千吨货物经由瓜达尔港湾运送各地，中白工业园不断吸引国际合作项目，阿尔及利亚一号卫星为当地通信昼夜服务……共建“一带一路”倡议思想之深邃、行动之务实，打造了今天逐渐完善的“六廊六路多国多港”总框架，塑造了中国与共建国家“硬联通”的格局，创造了各国人民之间“心联通”的基础，让“一带一路”的“朋友圈”越来越大，迈向和平发展、合作共赢道路的脚步愈加坚实。

六廊六路

正所谓“道路通，百业兴”，第三届“一带一路”国际合作高峰论坛开幕式主旨演讲中提及，“奔行在铁路上的列车，驰骋在公路上的汽车，联通各国的空中航班，劈波斩浪的货轮，快捷方便的数字电商，成为新时代国际贸易的驼铃、帆影”。

“一带一路”所着力构建的，就是以经济走廊为引领，以大通道和信息高速公路为骨架，以铁路、公路、机场、港口、管网为依托，涵盖陆、海、天、网的全球互联互通网络。由此，让我们以“一带一路”陆路运输与海上运输的标志性项目为回望的基点，回顾“一带一路”所贯彻践行的互利共赢之路。

02

陆路运输：穿越万里的“钢铁大动脉”

十年建设以来，“一带一路”合作从“大写意”进入“工笔画”阶段，把规划图转化为实景图，一批批陆路运输的标志性项目于沿线各国落地生根，共同构成亚欧大陆上穿越万里的“钢铁大动脉”，为“一带一路经济带”的交通运输发展提供源源不断的动力和支持。

🚆 往来不息的“钢铁驼队”——“中欧班列”

中欧班列是是依托亚欧大陆桥，按照固定线路、班期开行，运行于中国与欧洲以及“一带一路”沿线相关国家间的铁路国际联运列车。

作为共建“一带一路”的旗舰项目和标志性品牌，中欧班列的开行列数从运行之初的不到20列发展为2022年的1.6万列。其间数百倍的增长展现了这支“钢铁驼队”惊人的发展空间。10年来，中欧班列累计开行7.7万列，通达欧洲25个国家的217个城市，已成为国际经贸合作的重要桥梁。

织线成网，驰而不息，共建城市作为“一带一路”建设的重要节点，在中欧班列这组“黄金列车”的连接下，既是共建者又是受益者。若干中国城市亦积极参与其中，成都、西安、沈阳等城市相继成为中欧班列的始发站，其星罗棋布的地理位置让遍及全国的城市枢纽连点成面，共同构成中欧班列多线齐发的“强联通”贸易格局。

以义乌这座向来有享有“小商品之都”美称的城市为例，义乌成为中欧班列的战略支点，一方面让“义新欧”的多条中欧班列线路得以开设，进一步强化了中欧班列的贸易通道属性，另一方面也为城市本身带来了经济、文化等方面的溢出效应，使义乌成为通达四海、买卖全球的“世界货地”。

🚆 开放合作的“黄金路”——中老铁路

一条铁路连接两国多地，2021年12月3日，历时5年修建的中老铁路正式通车，这标志着中国与老挝两国的互联互通实现了跨越式的发展，“一带一路”又一项惠民工程在共商共建共享原则下顺利落地。中老铁路作为老挝“变陆锁国为陆联国”战略对接的重要项目，一经开通就承担起跨境运输的重要物流任务。

两年来，其运输货物已逐渐由初期的化肥、百货扩展至电子产品、光伏产品、冷链水果等100多种品类，运营时速达160公里，实现了对泰国、缅甸、马来西亚等多个周边国家和地区的辐射。中老铁路实现了老挝人民感受现代化铁路的梦想，其不断增长的货运量和客运量也为老挝的社会进步和经济发展注入新的活力。

中老铁路开通以来，货运和客运量持续增长

中老铁路是共建“一带一路”倡议提出后，首条以中方为主投资建设、全线采用中国技术标准、使用中国设备并与中国铁路网直接联通的国际铁路。由于建设体量庞大、沿线地质地貌复杂，截至正式通车时，中老铁路全程共建有167座隧道、301座桥梁，真正意义上做到了为惠及民生而“逢山开路、遇水架桥”。

如今，这条开放合作的“黄金路”，仍在以十足的发展潜力进行拓展，为越来越多连通区域带去发展新契机，展现了中国帮助“一带一路”沿线国家和地区破解条件局限、走向发展之路的命运共同体精神。

03

海路运输：贯连古今的“海上新丝路”

海上丝绸之路兴起于秦汉，鼎盛于宋元，其来往范围覆盖了大半个地球，是古代东西方文化经济交流的重要通道之一。2013年10月，习近平总书记在出访东南亚国家期间提出共同建设“21世纪海上丝绸之路”，让这条贯连古今的海上丝路在新时代又焕发出勃勃生机。近十年来，海上丝绸之路的进出口贸易指数（以2015年3月为基期，基点为100）总体呈现上升趋势，可见“海上新丝路”为促进全球贸易发展提供了新的平台。

🚢 “世界第一大港”——宁波舟山港

在中国大陆的海岸线中部、“丝绸之路经济带”和“21世纪海上丝绸之路”的交汇点上，“世界第一大港”宁波舟山港坐落于此，其货物吞吐量连续十四年位居世界第一，连接着超过200个国家和地区的600多个港口。为深度融入“一带一路”，这十年来宁波舟山港持续加大对共建国家航线航班的开发力度，如今“一带一路”航线已增至129条，“一带一路”2022年箱量达1320万标准箱，二者均占港口航线、箱量总量的四成以上。

宁波舟山港是中国对外开放的一类口岸，具有连接中外的天然属性，近年来宁波舟山港累计与30多个“一带一路”共建国家港口建立了友好港、姊妹港关系，不断扩大的“朋友圈”反映出宁波舟山港与世界港航伙伴的“心联通”，是促进港航命运共同体的生动实践。

此外，为提高“一带一路”运输效能，宁波舟山港已开通100条集装箱海铁联运线路，业务范围涵盖16个省（市、自治区）、64个地级市，基本形成了北接古丝绸之路、中汇长江经济带、南联千里沪昆线的三大物流通道，让海上丝路的繁荣贯通古今。

🚢 “扼守通道之地”——比雷埃夫斯港

比雷埃夫斯，在希腊语中意为“扼守通道之地”，作为世界历史最悠久的港口之一，比雷埃夫斯港具有得天独厚的水文条件，是一带一路“蓝色经济通道”上不可或缺的深水良港，也是中国企业在海外全资拥有的第一家港口。

2008年6月，中国远洋运输集团旗下公司中标比港二号、三号集装箱码头35年特许经营权，自2009年10月负责经营；2016年，中远海运集团完成对比雷埃夫斯港务局（PPA）67%股权的收购，接手一号码头。

从正式接手比雷埃夫斯港起，中远海运集团就对其进行了一系列的改造升级，使这座古老港口迎来扭亏为盈的“新生”。到2022年，比雷埃夫斯港口公司的每股分红已经上涨至2016年收购之前的十倍，其营业总额和净利润也在不断实现攀升。

当前，比雷埃夫斯港已成为中欧陆海快线的起点，是中希共建“一带一路”的典范项目，在地中海、欧洲乃至全球航运格局中占据日益重要的位置。

04

“一带一路”：构建人类命运共同体重大实践

协和万邦，“一带一路”引领区域协同发展合作共赢。十年间，“一带一路”这一长周期、跨国界、系统性的世界工程、世纪工程给世界带来了引人注目的深刻变化，吸引了大量认同中国主张的同道者偕行。截至2023年10月，中国已与152个国家和地区签署了共建“一带一路”合作协议，不断扩大的“朋友圈”充分表明，“一带一路”是着眼人类前途命运、顺应国际社会期待的互利共赢之路。

立己达人，“一带一路”构建人类命运共同体重大实践。“一带一路”走过的十年让古丝绸之路的盛景在新时代复苏，陆上“钢铁驼队”使“商旅不绝于途”，海上“重重帆影”再现“舶交海中，不知其数”，陆海并行的互联互通格局让共建国家共享成果。根据世界银行测算，到“一带一路”框架下交通建设项目全部实施，全球收入有望在2030年增加0.7%-2.9%，使760万人摆脱极端贫困、3200万人摆脱中度贫困。
